# GitLab AI Hackathon — Overview

> Source: https://gitlab.devpost.com/

## Theme: "You Orchestrate. AI Accelerates."

- **Deadline**: March 25, 2026 @ 2:00pm EDT
- **Total Prizes**: $65,000 in cash
- **Participants**: 3,604+
- **Tags**: Databases, Machine Learning/AI, Productivity

## Core Message

> AI writes code. That's expected now. The real opportunity? Everything else.
> Planning. Security. Compliance. Deployments. These are the bottlenecks that slow teams down every day.

## Why Join?

1. **Be early**: Access GitLab Duo Agent Platform while it's fresh — learn, build, shape it
2. **Learn agents**: Build agents that assist with tedious tasks (creating tests, etc.)
3. **Learn flows**: Orchestrate an entire team of purpose-built agents (build, test, security scan, deploy)
4. **Make an impact**: Your agent could ship to thousands of GitLab users
5. **Win $65,000 in prizes**

## What to Build

> Anything that removes friction from the software development lifecycle.
> - Automate security fixes
> - Streamline code reviews
> - Generate compliance reports
> - Flag risks before they become problems

**Key Requirements**:
- Tell a story: here's the pain → here's how the agent solves it → here's what changes
- **Chat alone won't qualify** — agents must react to triggers and take action
- At least one custom public agent or public flow must be created

## Submission Checklist

1. URL to project in [GitLab AI Hackathon group](https://gitlab.com/gitlab-ai-hackathon) (public, MIT license, all source code)
2. Text description of features and functionality
3. Demo video (3 min max, on YouTube/Vimeo, public)

## Judging Criteria (equally weighted)

1. **Technological Implementation** — Code quality, leverages Tools/Triggers/Context
2. **Design & Usability** — Easy to install/configure/interact, workflow makes sense
3. **Potential Impact** — Solves real "AI Paradox" bottleneck (planning, security, operations)
4. **Quality of the Idea** — Creative, unique, improves on existing concepts
