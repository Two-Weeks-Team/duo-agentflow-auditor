## Inspiration — Example Projects

### Achievable Projects
- **Journey 1: VueJS Unit Test Writer** — Agent that generates unit tests for Vue.js components
- **Journey 4: Issue Triage** — Flow of agents that categorizes and prioritizes issues automatically
- **Journey 5: Documentation Writer** — Agent that creates documentation using Knowledge Graph
- **Journey 6: Security Sentinel** — CLI agent that monitors and flags security concerns
- **Journey 7: OpenTofu Expert** — Agent that helps with Infrastructure as Code
- **Journey 8: Jenkins Migration** — Agent that helps migrate Jenkins pipelines to GitLab CI

### Aspirational Projects
- **Journey 2: IDE-First CI/CD Configuration** — Configure pipelines without leaving editor
- **Journey 3: Pipeline Observability** — Flow of agents that monitors and explains pipeline behavior

## Category Prize Details

### GitLab + Anthropic ($13,500 total)
- Agents run within GitLab have access to Anthropic Models by default
- Cannot change model in sandbox
- See: [Agent Platform AI Models](https://docs.gitlab.com/user/duo_agent_platform/model_selection/)

### GitLab + Google Cloud ($13,500 total)
- Use [Google Cloud integration](https://docs.gitlab.com/ci/gitlab_google_cloud_integration/)

### Green Agent ($3,000)
- Build agent/flow that advances **sustainability in software development**
- Measure, understand, reduce environmental impact
- Track LLM energy consumption, optimize CI/CD for efficiency, provide sustainability insights
